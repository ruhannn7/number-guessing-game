# number-guessing-game


A simple Python console game where the player must guess a randomly generated number between 1 and 20 within 5 attempts.

## How to Run
1. Install Python 3.
2. Download the repository files.
3. Run this command:
    python main.py

-Rules
  The number is between 1 and 20
  You get 5 attempts
  Type 'q' to quit the game at any time
  Hints: “Too high” / “Too low”
