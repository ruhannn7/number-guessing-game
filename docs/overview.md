# Project Overview

This project is a simple console-based Number Guessing Game written in Python.

The program:
- Selects a random number between **1 and 20**
- Gives the player **5 attempts** to guess it
- Displays hints like “Too high” or “Too low”
- Allows the player to quit anytime by typing 'q'

The project demonstrates:
- Basic Python logic
- Input handling
- Loops & conditions
- User interaction through console
